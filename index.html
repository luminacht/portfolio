Classification of Emergency Type Texts using Machine Learning Algorithms

  • Performed a research study on machine learning and deep learning algorithms to find the most suitable for classifying Bicol, Tagalog, and English emergency texts in Legazpi City.
  • Analyzed existing emergency texts collected from Legazpi City Disaster Risk Reduction and Management Office (LDRRMO), which was used for training the text classification models.
  • SVM, Naive Bayes, and Random Forest were trained and compared after researching and testing different machine and deep learning algorithms.
  • Deployed the best performing model to an android-based mobile application prototype using TFLite.

8-Puzzle Solver using IDS and A* Search

A C++ implementation to the 8-Puzzle problem which uses A* Search and Iterative Deepening Search (IDS). IDS is a blind search strategy whilst A* is an informed search strategy which uses a heuristic function.

A given goal state is declared where the following input states should reach the said goal using both search strategies.The program returns the number of nodes expanded, cost of the solution path, the path taken from the initial state to the goal state, and the runtime of the program in milliseconds.
